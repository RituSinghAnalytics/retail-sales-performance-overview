# Retail Sales Performance Overview

Power BI dashboard for retail sales performance analysis.

## Overview
This project analyzes retail sales performance across category, gender, and monthly trends. It provides a clean executive view of key business metrics and supports data-driven decision-making.

## Tools Used
- Power BI
- DAX
- Data Modeling
- Data Visualization

## Key Features
- KPI cards for Total Revenue, Average Revenue, and Units Sold
- Category-wise customer distribution
- Revenue split by gender
- Sales trend by category
- Monthly revenue by category

## Business Insights
- Identifies which product categories contribute most to revenue.
- Highlights customer distribution across categories.
- Shows monthly sales movement for trend analysis.
- Supports quick executive-level reporting.
